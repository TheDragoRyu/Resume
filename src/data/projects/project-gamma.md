## Problem

Code review bottlenecks slowed down the development cycle, with reviewers spending hours on style and pattern issues.

## Approach

Created a Python CLI that hooks into git workflows, analyzes diffs, and provides AI-powered suggestions for code quality improvements, security issues, and performance optimizations.

## Results

- Cut average review time by 35%.
- Caught 20+ security issues in the first month.
- 200+ stars on GitHub.

## Tech Stack

Python, OpenAI API, Git, Click (CLI framework)
