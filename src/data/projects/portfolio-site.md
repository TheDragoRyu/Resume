## Problem
Needed a portfolio that could showcase projects, skills, and experience in an engaging, interactive format without relying on a CMS or backend.

## Solution
Built as a fully static Next.js site, the portfolio presented a 3D visualization alongside dedicated resume and projects sections. Each project received its own page generated from structured content. The site was designed to load without a server and render consistently across environments.

## Highlights
- Interactive 3D scene rendered in the browser as a visual focal point
- Dedicated project detail pages reachable from a projects listing
- Keyboard-navigable layout with screen reader support throughout
- Resume page presenting work history and skills in structured form
- Contact section accessible via a direct link from the main navigation

## Tech Stack
Next.js 15, React 19, TypeScript, Three.js, React Three Fiber, Tailwind CSS 4
